O que é uma Matriz?

Uma matriz é uma tabela retangular de números (ou símbolos) organizados em linhas e colunas. Uma matriz com m linhas e n colunas chama-se matriz m×n.

Notação geral
A = [aᵢⱼ] onde i = 1..m (linha), j = 1..n (coluna)
aᵢⱼ representa o elemento na linha i e coluna j.

Terminologia Essencial

  • Dimensão / Ordem: dada por m×n (linhas × colunas). Ex: 3×4 tem 3 linhas e 4 colunas.
  • Elemento aᵢⱼ: valor na linha i, coluna j. Em A = [[1,2],[3,4]], a₁₂ = 2.
  • Diagonal principal: elementos a₁₁, a₂₂, ..., aₙₙ (só em matrizes quadradas).
  • Diagonal secundária: elementos a₁ₙ, a₂,ₙ₋₁, ..., aₙ₁.
  • Traço (tr A): soma dos elementos da diagonal principal.
  • Submatriz: matriz obtida removendo linhas e/ou colunas de A.
  • Menor: determinante de uma submatriz quadrada de A.
A notação aᵢⱼ usa sempre primeiro a linha e depois a coluna — pensa em "ir à linha primeiro, depois à coluna" como num mapa de coordenadas.

Por Forma / Dimensão

Matriz Linha 1×n

Tem apenas 1 linha e n colunas.

Tem m linhas e apenas 1 coluna.

Mesmo número de linhas e colunas. Base da maioria das propriedades avançadas.

A matriz identidade é simultaneamente diagonal, escalar, simétrica, ortogonal, idempotente e involutória — é o exemplo mais "rico" em propriedades.

Adição e Subtração

Só é possível entre matrizes de mesma dimensão. Soma-se elemento a elemento.

Definição
(A + B)ᵢⱼ = aᵢⱼ + bᵢⱼ
Comutativa: A+B = B+A  |  Associativa: (A+B)+C = A+(B+C)
Não é possível somar matrizes de dimensões diferentes. A soma de uma 2×3 com uma 3×2 é indefinida.

Multiplicação por Escalar

Multiplica cada elemento da matriz pelo número (escalar) k.

Definição
(k·A)ᵢⱼ = k · aᵢⱼ
Distributiva: k(A+B) = kA+kB  |  (k+m)A = kA+mA

Multiplicação de Matrizes

Para multiplicar A (m×p) por B (p×n), o número de colunas de A tem de ser igual ao número de linhas de B. O resultado C tem dimensão m×n.

Definição
cᵢⱼ = Σₖ aᵢₖ · bₖⱼ  (k de 1 a p)
Cada elemento cᵢⱼ é o produto escalar da linha i de A pela coluna j de B.
A multiplicação NÃO é comutativa! Em geral, A·B ≠ B·A. Mas é associativa: (A·B)·C = A·(B·C).
Regra prática: (m×p) · (p×n) = (m×n). As dimensões "internas" cancelam-se e ficam as "externas".

Transposta

A transposta Aᵀ troca linhas por colunas: a linha i de A torna-se a coluna i de Aᵀ.

Definição
(Aᵀ)ᵢⱼ = aⱼᵢ
(A+B)ᵀ = Aᵀ+Bᵀ  |  (A·B)ᵀ = Bᵀ·Aᵀ  |  (Aᵀ)ᵀ = A
Nota que na transposta do produto, a ordem inverte: (A·B)ᵀ = Bᵀ·Aᵀ. Este é um erro comum!

Inversa

A inversa A⁻¹ de uma matriz quadrada A existe se e só se det(A) ≠ 0 (matriz não-singular). Verifica: A·A⁻¹ = A⁻¹·A = I.

Para matrizes 2×2
A⁻¹ = (1/det A) · [ d −b ]
                    [ −c a ]
onde A = [[a,b],[c,d]] e det A = ad − bc
Para matrizes n×n — Método de Gauss-Jordan
[A | I] → operações elementares → [I | A⁻¹]
Forma a matriz aumentada e aplica eliminação de Gauss-Jordan.
  • (A·B)⁻¹ = B⁻¹·A⁻¹ — a ordem inverte (igual à transposta).
  • (Aᵀ)⁻¹ = (A⁻¹)ᵀ — transposta da inversa = inversa da transposta.
  • det(A⁻¹) = 1/det(A)

Potência de Matrizes

Só definida para matrizes quadradas. Aⁿ = A·A·...·A (n vezes).

  • A⁰ = I (convenção)
  • A⁻ⁿ = (A⁻¹)ⁿ (potências negativas usam a inversa)
  • Aᵐ·Aⁿ = Aᵐ⁺ⁿ
  • (Aᵐ)ⁿ = Aᵐⁿ

O que é o Determinante?

O determinante é um número escalar associado a qualquer matriz quadrada. Geometricamente, representa o fator de escala de área (2D) ou volume (3D) da transformação linear definida pela matriz.

  • det(A) = 0 → matriz singular (não tem inversa; as linhas são linearmente dependentes — pelo menos uma linha é combinação linear das outras)
  • det(A) ≠ 0 → matriz não-singular / invertível (tem inversa; o sistema A·x = b tem solução única)
  • |det(A)| = fator pelo qual a transformação escala áreas ou volumes
  • det(A) < 0 → a transformação inverte a orientação (reflexão)
Pensa assim: se pegarmos num quadrado de área 1 e o transformarmos com A, a área resultante é |det(A)|. Se det = 0, o quadrado "achata" numa linha — perde uma dimensão e deixa de ter área.

Casos Base: Ordem 1×1 e 2×2

Matriz 1×1
det([a]) = a
O determinante de uma matriz com um só elemento é esse elemento.
Matriz 2×2
A = [ a b ] det(A) = ad − bc [ c d ]
Multiplica a diagonal principal (↘) e subtrai o produto da diagonal secundária (↗). É a fórmula mais fundamental de todas.

Exemplo: A = [[3, 2], [1, 4]] → det(A) = 3·4 − 2·1 = 12 − 2 = 10

Regra de Sarrus — Ordem 3×3

A Regra de Sarrus é um mnemónico visual para calcular o determinante de matrizes 3×3. Foi proposta pelo matemático francês Pierre Frédéric Sarrus em 1833. Atenção: só funciona para 3×3!

Fórmula geral 3×3
[ a b c ] A = [ d e f ] [ g h i ] det(A) = (aei + bfg + cdh) − (ceg + afh + bdi)
Soma dos 3 produtos de diagonais "para baixo" (↘) menos a soma dos 3 produtos de diagonais "para cima" (↗).

Como aplicar o método visual:

  1. Escreve a matriz 3×3 normalmente.
  2. Repete as duas primeiras colunas à direita da matriz (fica uma grelha 3×5).
  3. Soma os produtos das 3 diagonais que descem da esquerda para a direita (↘): a·e·i, b·f·g, c·d·h.
  4. Subtrai os produtos das 3 diagonais que sobem da esquerda para a direita (↗): c·e·g, a·f·h, b·d·i.
Exemplo numérico
A = [ 1 2 3 ]
    [ 4 5 6 ]
    [ 7 8 9 ]
Diagonais ↘:
  1·5·9 = 45
  2·6·7 = 84
  3·4·8 = 96
  Soma = 225
Diagonais ↗:
  3·5·7 = 105
  1·6·8 = 48
  2·4·9 = 72
  Soma = 225
det(A) = 225 − 225 = 0 (as linhas são linearmente dependentes — a linha 3 é L1 + 2·(L2−L1))
⚠️ A Regra de Sarrus NÃO se generaliza para ordens superiores a 3. Para 4×4 ou maior, é obrigatório usar a Expansão de Laplace ou outro método.

Expansão de Laplace (Cofatores) — Qualquer Ordem

A Expansão de Laplace, também chamada desenvolvimento por cofatores, é o método geral para calcular determinantes de qualquer ordem. Foi formalizada pelo matemático francês Pierre-Simon Laplace no século XVIII.

A ideia central é reduzir um determinante de ordem n ao cálculo de n determinantes de ordem n−1, e estes a n−1 de ordem n−2, e assim recursivamente até chegar a determinantes 2×2 que sabemos calcular diretamente.

Definições essenciais
Menor complementar Mᵢⱼ = det(submatriz sem linha i e coluna j) Cofator Cᵢⱼ = (−1)^(i+j) · Mᵢⱼ Expansão pela linha i: det(A) = aᵢ₁·Cᵢ₁ + aᵢ₂·Cᵢ₂ + ... + aᵢₙ·Cᵢₙ Expansão pela coluna j: det(A) = a₁ⱼ·C₁ⱼ + a₂ⱼ·C₂ⱼ + ... + aₙⱼ·Cₙⱼ
Podes expandir por qualquer linha ou coluna — o resultado é sempre o mesmo! Escolhe a que tiver mais zeros para simplificar o cálculo.
Grelha de sinais (−1)^(i+j)

O fator de sinal alterna como um tabuleiro de xadrez. Para uma matriz 3×3:

[ + − + ]
[ − + − ]
[ + − + ]

Para qualquer n×n: posição (i,j) tem sinal + se i+j for par, sinal − se i+j for ímpar.

Exemplo: det de matriz 3×3 pela linha 1
A = [ 2 1 3 ]
    [ 0 4 1 ]
    [ 5 2 6 ]
Expansão pela linha 1:
det(A) = a₁₁·C₁₁ + a₁₂·C₁₂ + a₁₃·C₁₃

C₁₁ = (+1) · det([[4,1],[2,6]]) = +(4·6 − 1·2) = +22
Contribuição: 2 · 22 = 44

C₁₂ = (−1) · det([[0,1],[5,6]]) = −(0·6 − 1·5) = −(−5) = +5
Contribuição: 1 · 5 = 5

C₁₃ = (+1) · det([[0,4],[5,2]]) = +(0·2 − 4·5) = −20
Contribuição: 3 · (−20) = −60

det(A) = 44 + 5 + (−60) = −11
Estratégia: escolhe a linha/coluna com mais zeros

Considera a matriz:

B = [ 3 0 0 2 ]
    [ 0 5 0 0 ]
    [ 1 0 4 0 ]
    [ 0 0 2 7 ]

A linha 2 tem três zeros — expandir por ela significa calcular apenas 1 cofator em vez de 4:
det(B) = 5 · C₂₂ + 0 + 0 + 0 = 5 · C₂₂
Isto reduz imenso o trabalho!

💡 Resumo prático: Para matrizes 2×2 usa a fórmula direta. Para 3×3 usa Sarrus ou Laplace. Para 4×4 e maior, usa sempre Laplace (ou Eliminação de Gauss, que é mais eficiente computacionalmente). Nesta app, a secção Determinante mostra automaticamente Sarrus para 3×3 e Laplace para 4×4+.

Determinante por Eliminação de Gauss

Para matrizes grandes, o método mais eficiente é usar a Eliminação de Gauss para transformar A numa matriz triangular superior U, e depois multiplicar a diagonal.

Fórmula
det(A) = (−1)^s · u₁₁ · u₂₂ · · · uₙₙ
s = número de trocas de linhas durante a eliminação. Se não houve trocas, s = 0 e o fator é +1. Cada troca de linhas multiplica o determinante por −1.

Porquê funciona? As operações do tipo III (somar múltiplo de uma linha a outra) não alteram o determinante. As operações do tipo I (troca de linhas) multiplicam o det por −1. Portanto, após a eliminação, basta contar as trocas e multiplicar a diagonal de U.

Propriedades do Determinante

  • det(I) = 1 — a identidade tem determinante 1
  • det(Aᵀ) = det(A) — a transposta não altera o determinante
  • det(A·B) = det(A) · det(B) — multiplicatividade
  • det(kA) = kⁿ · det(A) — para matriz n×n, cada linha é multiplicada por k
  • det(A⁻¹) = 1/det(A) — pois det(A)·det(A⁻¹) = det(I) = 1
  • Trocar duas linhas muda o sinal do determinante
  • Se duas linhas forem iguais ou proporcionais, det = 0
  • Multiplicar uma linha por k multiplica det por k
  • Somar o múltiplo de uma linha a outra não altera det
  • Se uma linha for toda zeros, det = 0
det(A·B) = det(A)·det(B) é muito útil: det(A²) = det(A)², det(A³) = det(A)³, etc. E se A for ortogonal (Aᵀ = A⁻¹), então det(A) = ±1.

Condensação de Dodgson

Método inventado por Lewis Carroll (Charles Dodgson, autor de Alice no País das Maravilhas!) em 1866. Calcula o determinante por condensação iterativa, reduzindo a matriz passo a passo usando determinantes 2×2.

Fórmula de condensação
M^(k+1)ᵢⱼ = (M^k_ᵢⱼ · M^k_ᵢ₊₁,ⱼ₊₁ − M^k_ᵢ,ⱼ₊₁ · M^k_ᵢ₊₁,ⱼ) / M^(k-1)_ᵢ₊₁,ⱼ₊₁
A cada iteração, a matriz (n×n) reduz para (n−1)×(n−1). O denominador é o elemento interior da iteração anterior. Falha se algum elemento interior for zero (requer perturbação ou troca de linhas).
⚠️ A Condensação de Dodgson requer cuidado quando surgem zeros nos elementos interiores. Nesse caso é necessário reorganizar a matriz antes de continuar.

Representação Matricial

Um sistema de equações lineares pode ser escrito de forma compacta como A·x = b, onde A é a matriz dos coeficientes, x o vetor incógnita e b o vetor dos termos independentes.

Sistema → Forma matricial
2x + 3y = 7 [ 2 3 ] [x] [7]
x − y = 1 → [ 1 -1 ] [y] = [1]

A · x = b

Classificação dos Sistemas (Teorema de Rouché-Capelli)

  • Sistema Possível e Determinado (SPD): rank(A) = rank(A|b) = n → uma solução única.
  • Sistema Possível e Indeterminado (SPI): rank(A) = rank(A|b) < n → infinitas soluções (n − rank graus de liberdade).
  • Sistema Impossível (SI): rank(A) ≠ rank(A|b) → sem solução.
Para verificar: forma a matriz aumentada [A|b] e calcula o rank. Se rank(A) = rank(A|b) = n, o sistema tem solução única dada por x = A⁻¹·b.

Métodos de Resolução

Eliminação de Gauss

Reduz [A|b] à forma escalonada por operações elementares nas linhas. Depois resolve por substituição retroativa.

Gauss-Jordan

Continua a eliminação até à forma escalonada reduzida. Lê directamente as soluções sem substituição.

Regra de Cramer

xᵢ = det(Aᵢ)/det(A), onde Aᵢ é A com a coluna i substituída por b. Só funciona para SPD (n equações, n incógnitas).

Inversão (x = A⁻¹·b)

Se A é invertível, a solução é x = A⁻¹·b. Útil quando há múltiplos vetores b para o mesmo A.

Operações Elementares nas Linhas

  • Tipo I — Troca: Lᵢ ↔ Lⱼ (troca duas linhas). Muda o sinal do determinante.
  • Tipo II — Escala: Lᵢ ← k·Lᵢ (multiplica linha por escalar k≠0). Multiplica det por k.
  • Tipo III — Eliminação: Lᵢ ← Lᵢ + k·Lⱼ (soma múltiplo de outra linha). Não altera o determinante.
Na eliminação de Gauss para resolver sistemas, as operações elementares não alteram o conjunto solução — apenas simplificam o sistema.

Rank (Posto) de uma Matriz

O rank de uma matriz é o número de linhas (ou colunas) linearmente independentes — equivale ao número de pivôs na forma escalonada.

  • rank(A) ≤ min(m, n)
  • rank máximo (= min(m,n)) → matriz de rank completo
  • rank(A) = rank(Aᵀ)
  • rank(A·B) ≤ min(rank A, rank B)
  • Nullity: n − rank(A) = dimensão do núcleo (espaço nulo)
  • Teorema Rank-Nulidade: rank(A) + nullity(A) = n (número de colunas)

Traço

O traço de uma matriz quadrada é a soma dos elementos da diagonal principal.

Definição e propriedades
tr(A) = Σᵢ aᵢᵢ = a₁₁ + a₂₂ + ... + aₙₙ
tr(A+B) = tr(A)+tr(B)  |  tr(kA) = k·tr(A)  |  tr(Aᵀ) = tr(A)  |  tr(A·B) = tr(B·A)
O traço é igual à soma dos valores próprios (eigenvalues) de A — uma ligação profunda à álgebra linear avançada.

Norma de Frobenius

A norma de Frobenius mede a "magnitude" de uma matriz, generalizando o conceito de comprimento de vetor.

Definição
‖A‖_F = √(Σᵢⱼ aᵢⱼ²)
Raiz quadrada da soma dos quadrados de todos os elementos. Análoga à norma euclidiana de vetores.

Valores Próprios (Eigenvalues)

Um escalar λ é valor próprio de A se existe um vetor não-nulo v (vetor próprio) tal que A·v = λ·v. O vetor é apenas escalado, sem rotação.

Equação característica
det(A − λI) = 0
Resolver este polinômio de grau n dá os n valores próprios (podem ser complexos).
  • Soma dos valores próprios = tr(A)
  • Produto dos valores próprios = det(A)
  • Se A é simétrica, todos os eigenvalues são reais
Valores próprios são fundamentais em: análise de estabilidade, compressão de dados (PCA), vibrações mecânicas, PageRank do Google, e muito mais.

Diagonalização

Uma matriz A é diagonalizável se existe uma matriz invertível P tal que P⁻¹·A·P = D, onde D é diagonal com os eigenvalues na diagonal.

Aplicação: potências eficientes
Aⁿ = P · Dⁿ · P⁻¹
Como Dⁿ só eleva os elementos diagonais, calcular Aⁿ torna-se muito mais eficiente.
Nem todas as matrizes são diagonalizáveis. Uma matriz n×n é diagonalizável se e só se tem n vetores próprios linearmente independentes.

Formas Quadráticas e Definição

Uma forma quadrática é xᵀ·A·x. A matriz A (simétrica) classifica-se em:

  • Definida positiva: todos eigenvalues > 0 (xᵀAx > 0 para todo x≠0)
  • Semi-definida positiva: todos eigenvalues ≥ 0
  • Definida negativa: todos eigenvalues < 0
  • Indefinida: eigenvalues mistos (positivos e negativos)
Estas classificações são essenciais em otimização: um ponto crítico de uma função é mínimo se a Hessiana (matriz das segundas derivadas) for definida positiva nesse ponto.
