O que é o Matrix Lab

O Matrix Lab é uma ferramenta educativa interativa criada para quem estuda álgebra linear e precisa de um ambiente visual para explorar operações com matrizes. Em vez de apenas mostrar o resultado final, a aplicação detalha cada passo do processo — tornando os algoritmos transparentes e fáceis de acompanhar.

O projeto foi desenvolvido com HTML, CSS e JavaScript puro, sem nenhuma biblioteca externa. Toda a lógica matemática, desde a eliminação de Gauss até à condensação de Dodgson e resolução de equações matriciais, foi implementada de raiz para garantir controlo total sobre cada operação.

O objetivo principal é simples: tornar a álgebra linear menos intimidante. Ver cada operação elementar — trocas de linhas, eliminações, escalamentos — acontecer passo a passo ajuda a construir intuição matemática sólida.

Como funciona por dentro
1

Introdução de dados

Define as dimensões com os botões +/− e preenche os valores. O botão Aleatório gera valores inteiros para testar. Usa ponto como separador decimal (ex: 3.14).

2

Validação e compatibilidade

A aplicação verifica a compatibilidade das dimensões e deteta erros — como colunas de A ≠ linhas de B, ou matriz singular na inversa.

3

Cálculo com registo de passos

Cada algoritmo grava cada operação elementar — trocas de linhas, fatores de eliminação, escalamentos — apresentado de forma visual e legível.

4

Apresentação animada

Resultados aparecem célula a célula com animações. O modo frações pode ser ativado para resultados exatos em vez de decimais.

Conceitos matemáticos

A multiplicação de matrizes não é comutativa — em geral, A×B ≠ B×A. O número de colunas de A deve ser igual ao número de linhas de B:

C[i,j] = Σ(k=1→n) A[i,k] · B[k,j]

A Eliminação de Gauss com pivotagem parcial reduz a matriz à forma triangular superior minimizando erros numéricos. O determinante é o produto da diagonal, com sinal ajustado por cada troca de linhas.

A Condensação de Dodgson (1866), proposta por Lewis Carroll, condensa a matriz iterativamente usando determinantes 2×2 até obter um único escalar — o determinante.

As equações matriciais como A·X = B resolvem-se com X = A⁻¹·B. Para X·B = C, tem-se X = C·B⁻¹. Para A·X·B = C, aplica-se X = A⁻¹·C·B⁻¹.
